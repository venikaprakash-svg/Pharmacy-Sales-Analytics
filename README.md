#  Pharmacy Sales Analytics using Excel, SQL & Python

## Project Overview
This project demonstrates an end-to-end healthcare data analytics workflow using pharmacy sales data. The objective is to analyze medicine sales, inventory levels, and category-wise performance using Excel, SQL, and Python.

## Tools & Technologies
- Microsoft Excel
- MySQL
- Python
- Pandas
- Matplotlib

## Dataset
The dataset contains 101 pharmacy sales records with the following fields:
- Medicine Name
- Category
- Quantity Sold
- Unit Price
- Total Sale
- Stock Remaining
- Date

## Analysis Performed
- Data cleaning and preparation in Excel
- Pivot Table analysis
- SQL database creation and querying
- Sales and inventory analysis using Python (Pandas)
- Data visualization using Matplotlib

## Visualizations
- Sales by Category (Bar Chart)
- Stock Remaining by Category (Bar Chart)
- Top 10 Medicines by Total Sales (Horizontal Bar Chart)

## Key Insights
- Identified top-performing medicines based on sales.
- Compared revenue across medicine categories.
- Analyzed inventory levels to identify low-stock categories.
- Demonstrated an end-to-end healthcare analytics workflow.
