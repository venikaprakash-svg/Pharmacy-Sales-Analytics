# Pharmacy Sales Analytics

## Project Overview
This project analyzes pharmacy sales data using Excel, SQL, and Python.

## Tools
- Microsoft Excel
- MySQL
- Python
- Pandas
- Matplotlib

## Project Workflow
1. Created pharmacy sales dataset
2. Cleaned and analyzed data using Excel
3. Imported data into MySQL
4. Performed SQL queries
5. Analyzed data using Python
6. Created business charts

## Charts
- Sales by Category
- Stock Remaining by Category
- Top 10 Medicines by Total Sales

## Skills
Excel | SQL | Python | Pandas | Data Analysis | Data Visualization
